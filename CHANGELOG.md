# Changelog

All notable changes to Super YouTube will be documented in this file.

## [1.0.0] - 2025-11-07

### Added
- Initial production release
- 21 fully functional features
- Video Player controls (8 features)
  - Loop Video Button
  - Loop Playlist Button
  - Volume Control with Mouse Scroll
  - Default Quality Settings
  - Default Playback Speed
  - Keyboard Shortcuts Fix
  - Real-time Audio Visualizer
  - Next/Previous Video Buttons
- Tools (6 features)
  - Screenshot Button
  - Picture-in-Picture Button
  - Skip Sponsors (SponsorBlock)
  - Disable Auto-Pause
  - Block End Screens
  - Auto-Expand Description
- Interface customization (7 features)
  - Classic Layout Theme
  - Hide Comments
  - Hide Live Chat
  - Hide Shorts Shelf
  - Redirect Shorts
  - Hide Shorts Button
  - Content Blocking (keywords/channels)

### Technical
- Manifest V3 compatibility
- Sandbox iframe protection
- Optimized code without excessive comments
- Chrome Web Store policy compliance
- Privacy policy included
- English language interface

### Security
- No data collection
- Local storage only
- Open source
- MIT License

## Version Format

This project uses [Semantic Versioning](https://semver.org/).

Format: MAJOR.MINOR.PATCH

- MAJOR: Incompatible API changes
- MINOR: New features (backward compatible)
- PATCH: Bug fixes (backward compatible)
