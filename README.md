# Super YouTube

Enhance your YouTube experience with 21 powerful features: Screenshot capture, Video loop, Content blocking, Audio visualizer, and much more!

## Features

### Video Player (8 features)
- Loop Video Button
- Loop Playlist Button  
- Volume Control with Mouse Scroll
- Default Quality Settings (up to 4K)
- Default Playback Speed
- Keyboard Shortcuts Fix
- Real-time Audio Visualizer
- Next/Previous Video Buttons

### Tools (6 features)
- Screenshot Button (high quality captures)
- Picture-in-Picture Button
- Skip Sponsors (SponsorBlock integration)
- Disable Auto-Pause
- Block End Screens
- Auto-Expand Description

### Interface (7 features)
- Classic Layout Theme
- Hide Comments
- Hide Live Chat
- Hide Shorts Shelf
- Redirect Shorts to Normal Player
- Hide Shorts Button
- Block Content by Keywords/Channels

## Installation

1. Download the extension
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `super-youtube` folder
6. Done! Access YouTube and enjoy!

## Privacy

- No data collection
- No tracking
- All settings stored locally
- Open source (MIT License)

## Usage

1. Click the extension icon in the toolbar
2. Enable the features you want
3. Click "Save Settings"
4. Enjoy your enhanced YouTube experience!
